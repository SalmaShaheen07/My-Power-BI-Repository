# My-Power-BI-Repository
## Project Summary
This repository contains my Power BI project completed during the Data Technician Bootcamp (May–June 2026).
The project focuses on transforming, modelling, and visualising retail and sales‑related datasets using Power BI Desktop and Power BI Service.

Across several labs and tasks, I developed skills in data cleaning, data modelling, DAX calculations, and interactive report design, culminating in professional‑quality dashboards suitable for business decision‑making.

## Project Overview
Throughout the Power BI module, I worked through hands‑on labs including:

Cleaning, transforming, and loading data using Power Query

Combining multiple data sources (Excel + OData feed)

Designing interactive reports with slicers, filters, and drill‑downs

Creating DAX calculated columns and measures

Building dashboards with a variety of visualisations

These tasks reflect real‑world analytics workflows used in business intelligence roles.

## Key Power BI Skills Demonstrated
### Data Cleaning & Transformation (Power Query)
Using Power Query, I performed essential data preparation tasks:

Removing errors and duplicates

Splitting and merging columns

Changing data types

Creating conditional columns

Filtering and shaping data for analysis

Combining datasets from multiple sources

This ensured the data model was clean, reliable, and ready for reporting.

### Data Modelling & DAX
I created a semantic model using:

Calculated columns (e.g., categorisation fields)

Measures using DAX, such as:

Total Sales

Average Sales

Year‑to‑Date (YTD) metrics

Percentage of Total

Relationships between tables for accurate cross‑filtering

These calculations enabled deeper insights and dynamic reporting.

### Interactive Report Design
I built interactive Power BI reports using:

Slicers (date, category, region, product)

Filters (visual‑level, page‑level, and report‑level)

Drill‑through pages

Tooltips for enhanced context

Bookmarks (where applicable)

These features allow users to explore the data from multiple angles.

### Visualisations Created
I used a wide range of Power BI visuals, including:

Bar & column charts – product performance, regional sales

Line charts – trends over time

Pie & donut charts – category distribution

Maps – geographic sales insights

Tables & matrices – detailed breakdowns

Cards & KPIs – high‑level metrics

These visuals helped turn raw data into clear, actionable insights.
